>> 01 / 04 · The map

Five categories of parachains

DEFI

Decentralised finance — exchanges, lending, stablecoins.

Examples: Acala (stablecoin + DEX), HydraDX (DEX), Bifrost (liquid staking).

SMART CONTRACTS

General-purpose smart contract execution. Developers deploy dApps.

Examples: Moonbeam (EVM), Astar (EVM + WASM).

PRIVACY

Privacy-preserving computation and data.

Examples: Phala Network (TEE computing).

GAMING + NFTs

Games and digital collectibles.

Examples: Ajuna Network (game logic), Unique Network (NFT chain).

INFRASTRUCTURE

Common goods parachains that serve the whole ecosystem.

Examples: Asset Hub (tokens), Bridge Hub (cross-chain bridges).

Not all parachains are equal. Some are "system chains" — run by Polkadot governance itself, serving the whole network. Others are independent teams building products.

>> 02 / 04 · DeFi

Decentralised finance on Polkadot

ACALA

The DeFi hub of Polkadot.

  • aUSD — a decentralised stablecoin pegged to USD
  • AcalaSwap — a decentralised exchange
  • Liquid DOT — stake DOT and get LDOT tokens you can still use in DeFi
  • Website: acala.network  ·  Token: ACA
HYDRADX

A next-generation DEX using an Omnipool.

  • All assets in one pool — lower fees and better prices than traditional AMMs
  • Formerly known as HydraDX
  • Website: hydradx.io  ·  Token: HDX
Why DeFi on Polkadot? Lower fees than Ethereum mainnet. Native cross-chain messages mean a DeFi protocol on Acala can interact with tokens from Moonbeam or Astar without a bridge.

>> 03 / 04 · Smart contracts

Where developers build dApps on Polkadot

MOONBEAM

Ethereum-compatible smart contract parachain.

  • Developers copy-paste Solidity contracts with zero changes
  • MetaMask, Hardhat, and Remix work out of the box
  • The easiest on-ramp for Ethereum devs
  • Website: moonbeam.network  ·  Token: GLMR
ASTAR NETWORK

Supports both EVM and WASM.

  • dApp staking — developers earn DOT just by building and deploying on Astar
  • Main dApp hub for Japan and Asia-Pacific
  • Website: astar.network  ·  Token: ASTR
For Ethereum developers, Moonbeam is the fastest path to Polkadot. For developers who want to go deeper (Rust/WASM contracts), Astar opens that door without leaving the ecosystem.

>> 04 / 04 · Infrastructure

The system parachains — serving the whole network

ASSET HUB

formerly Statemint

Creates and manages tokens for the whole Polkadot ecosystem. Like a token registry.

System chain — run by Polkadot governance, no separate token team.

BRIDGE HUB

Manages bridges to other networks — Ethereum, Bitcoin, Kusama.

Uses XCM to route bridged assets. Makes cross-chain movement safe and standardised.

COLLECTIVES

The on-chain home for the Polkadot Fellowship — the technical council of core developers.

Manages runtime upgrades and protocol decisions on-chain.

System chains are free for users — no separate token to buy, transactions paid in DOT. They're maintained by the network itself, not a startup.

>> The toolkit

Essential tools for the Polkadot ecosystem

POLKADOT.JS EXTENSION

Browser wallet for Polkadot.

Like MetaMask but for Polkadot/Substrate chains. Manages keys, signs transactions. Free, open source.

NOVA WALLET

Mobile wallet (iOS + Android).

User-friendly, supports staking and governance from your phone. The best beginner-friendly option.

SUBSCAN

Block explorer for all Polkadot ecosystem chains.

Search blocks, transactions, accounts. subscan.io

POLKADOT.JS APPS

Power user interface.

Staking, governance, technical operations. polkadot.js.org/apps
