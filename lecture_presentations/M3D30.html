>> 11:00 – 12:00 · Mini-project

Build your Tiny Blockchain

One Python file. All the functions. Working end to end.

hash_block(block)
SHA-256 fingerprint of all block fields
make_block(index, transactions, previous_hash, nonce)
Creates a valid block dict
mine_block(index, transactions, previous_hash, difficulty)
Finds valid PoW nonce
is_valid_chain(chain)
Returns True/False + error message
>> Required output

Your program must produce this output

demo.py
# When run, your file should: chain = [make_block(0, [], "0" * 64)] # genesis add_block(chain, [{"from":"Alice","to":"Bob","amount":10}]) add_block(chain, [{"from":"Bob","to":"Carol","amount":4}]) add_block(chain, [{"from":"Alice","to":"Dave","amount":2}]) for b in chain: print(f"Block {b['index']:2d} | nonce={b['nonce']:6d} " f"| hash={b['hash'][:16]}...") result, msg = is_valid_chain(chain) print(f"\nChain valid: {result} — {msg}")
Grading: Does it run? Does it produce valid linked blocks? Does validation return True? That's the core requirement.
