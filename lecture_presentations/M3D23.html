>> 02 / 03 · Building the block

make_block() — the blueprint

block.py
import hashlib, time def hash_block(block): content = ( str(block["index"]) + block["timestamp"] + str(block["data"]) + block["previous_hash"] ) return hashlib.sha256(content.encode()).hexdigest() def make_block(index, data, previous_hash): block = { "index": index, "timestamp": time.strftime("%Y-%m-%d %H:%M:%S"), "data": data, "previous_hash": previous_hash, "hash": "", # fill in next } block["hash"] = hash_block(block) return block
THE HASH IS LAST We fill every field first, then compute the hash. The hash is a fingerprint of all the other fields.
>> Inside hash_block()

How we fingerprint a block

To hash a block, we concatenate all its fields into one long string, then run SHA-256 on it.

hash_breakdown.py
block = { "index": 1, "timestamp": "2026-06-12 09:00:00", "data": "Alice sent 10 DOT to Bob", "previous_hash": "0000000000000000", } # The string we hash: content = "1" + "2026-06-12 09:00:00" + "Alice sent 10 DOT to Bob" + "0000000000000000" block["hash"] = hashlib.sha256(content.encode()).hexdigest() print(block["hash"]) # a 64-character hex fingerprint
Change anything in the block — the timestamp, one letter of the data — and hash_block() returns a completely different hash. That's the chain's tamper protection.
>> 03 / 03 · The first block

The genesis block — where every chain begins

The very first block is special: it has no previous block. So we invent a previous_hash of all zeros — a known "nothing" value.

genesis.py
genesis = make_block( index = 0, data = "Genesis Block", previous_hash = "0" * 64, # no previous block ) print(genesis) # { # "index": 0, # "timestamp": "2026-06-12 09:00:00", # "data": "Genesis Block", # "previous_hash": "0000...0000", # "hash": "a3f7..." # }
POLKADOT'S GENESIS

Polkadot's genesis block was created on 26 May 2020. Its hash and initial state were set by the network founders.

IMMUTABLE START

The genesis block can never be changed. Every subsequent block's chain of hashes ultimately links back to it.
