>> 01 / 04 · The tool

A block explorer = a search engine for the blockchain

Because blockchains are public, anyone can query the state directly. A block explorer is a website that does this for you — it reads the chain and displays blocks, transactions, and account balances in a readable format.

WHAT YOU CAN SEARCH FOR
  • Block number or hash → see all transactions in that block
  • Account address → see balance + full history
  • Transaction hash → see status, sender, receiver, amount
POLKADOT'S BLOCK EXPLORER
  • Subscan (subscan.io) is the most popular
  • Also: polkadot.js.org/apps has a built-in explorer
  • For Ethereum: Etherscan. Same concept, different network.
Why this matters: When you send DOT to someone, they don't have to trust you — they can verify on Subscan in seconds. No bank statement needed. The chain IS the statement.

>> 02 / 04 · The interface

A tour of polkadot.subscan.io

  1. Home page: live feed of recent blocks (left) and recent extrinsics (right). Updates every ~6 seconds.
  2. "Blocks" menu: paginated list of all blocks. Click any to see its contents.
  3. "Extrinsics" menu: all individual operations (transfers, staking, governance votes) across all blocks.
  4. Search bar: paste any block number, hash, address, or extrinsic hash.
  5. Network switcher (top right): change between Polkadot, Kusama, or specific parachains like Moonbeam.
Subscan covers all major Polkadot ecosystem chains. The same UI works for relay chain, parachains, and Kusama. The data format is identical because all use Substrate.

>> 03 / 04 · Blocks

Every field in a Subscan block

  • Block Number: 22,847,391 ← sequential block index (our "index" field)
  • Block Hash: 0x1a3f... ← SHA-256-like fingerprint of this block
  • Parent Hash: 0x9c82... ← hash of the previous block (our "previous_hash"!)
  • State Root: 0xef79... ← fingerprint of all account balances after this block
  • Extrinsics Root:0x5fee... ← fingerprint of all operations in this block
  • Block Time: 2026-06-12 09:00:06 ← when this block was produced
  • Validator: 5GrwvaEF5... ← who produced this block
  • Extrinsics: 4 ← how many operations are in this block
Parent Hash is our previous_hash. Click it on Subscan and it takes you to block 22,847,390. This is the exact chain-link you built in Python on Day 24.

>> What is an extrinsic?

Extrinsics — Polkadot's word for "transaction"

In Polkadot, an extrinsic is any external message sent to the chain — a transfer, a staking action, a governance vote. It's a broader term than "transaction."

A TRANSFER EXTRINSIC
  • Extrinsic Hash: 0xabcd...
  • Module: balances
  • Call: transfer_keep_alive
  • From: 5GrwvaEF5...
  • To: 5FHneW46...
  • Amount: 100 DOT
  • Fee: 0.0015 DOT
  • Status: Success
YOUR PYTHON TRANSACTION
{
  "from": "Alice",
  "to": "Bob",
  "amount": 10,
}

Same concept — source, destination, amount.
Polkadot adds: module, call, fee, status, signature.

Every extrinsic has a hash — you can look up any specific operation by its hash, the same way you look up a block by its hash.

>> Accounts

Looking up any address on Subscan

  1. Paste a Polkadot address (starts with "1" or "5") in the Subscan search bar
  2. See: free balance (available), reserved balance (locked for staking/deposits), total
  3. See: full transaction history — every DOT sent and received, with timestamp
  4. See: staking info — whether this address is a validator or nominator, stake amount
Privacy note: Addresses are pseudonymous. The address is public; the person behind it is not. But if you ever share your address publicly (to receive DOT), anyone can see your full transaction history.
EXPLORE Go to subscan.io → type the Polkadot Foundation's address: 13UVJyLnbVp9R… and see the history of one of the most active accounts on the network.
